\label{sec:experimental_setup}

\textbf{Hardware and software.} All reported measurements are produced on a
single NVIDIA RTX 3090 GPU with 24GB of VRAM. The implementation uses PyTorch,
Hugging Face Transformers, PEFT, BitsAndBytes, Datasets, and the Language Model
Evaluation Harness. The pipeline writes raw JSON snapshots and aggregated CSV
files under `results/single\_gpu\_3090/'.

\textbf{Model pool.} The planned formal pool contains nine locally deployable
open models, shown in \Cref{tab:model-overview}. The current result export is a
completed subset: public benchmark and efficiency rows are available for
Qwen3-0.6B, Qwen3-1.7B, Qwen3-4B, Qwen3-8B, Qwen2.5-7B-Instruct, and
DeepSeek-R1-Distill-Qwen-7B. Phi-3-mini, Yi-6B-Chat, and
Mistral-7B-Instruct-v0.3 remain in the planned pool but are not included in the
current public-benchmark result table.

\textbf{Railway-domain dataset.} The local corpus is stored under
`data/corpus/railway' and contains railway technical terminology and traction
power supply operation-and-maintenance documents. The generated dataset
contains 22,164 training examples, 2,770 validation examples, and 2,770 test
examples. Each JSONL record stores the prompt, expected answer, combined text,
category, and source.

\textbf{Benchmark suite.} Public benchmark tasks include MMLU, C-Eval, GSM8K,
and HumanEval. The local Domain QA task is included in the pipeline, but the
current aggregate CSV contains a one-sample exact-match sanity result rather
than the full 2,770-sample test evaluation. This distinction is preserved in
the result captions and discussion.

\textbf{Quantization settings.} Baseline inference is evaluated under bf16,
int8, and int4 settings when supported by the current stack. For each
model-precision pair, the pipeline records task scores, peak allocated VRAM,
peak reserved VRAM, latency, and generated-token throughput.

\textbf{QLoRA setup.} The QLoRA configuration uses 4-bit base loading, frozen
base weights, LoRA rank 64, LoRA alpha 16, dropout 0.05, learning rate
\(2\times10^{-4}\), three epochs, per-device train batch size 1, gradient
accumulation 16, and maximum sequence length 2,048. The current adapter export
contains Qwen3-4B and Qwen2.5-7B-Instruct.
